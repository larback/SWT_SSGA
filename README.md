# ssga-pmstr

Este repositório apresenta o código-fonte e as instâncias usadas no artigo "Algoritmo Genético de Estado Estável Aplicado à Minimização de Trocas de Ferramentas por Desgaste" submetido ao LVIII Simpósio Brasileiro de Pesquisa Operacional 2026.

Para compilar, utilize o comando "make" (certifique-se que o pacote build-essential está instalado em sua máquina). Em seguida, para executar, utilize o comando:

```cpp
  nohup ./run {pasta_com_suas_instancias} {número_de_execuçôes_independentes_por_instância} &
```

Qualquer dúvida ou pergunta podem ser enviados para o e-mail [lauracsrocha0@gmail.com](mailto:lauracsrocha0@gmail.com).

Observação: este código funciona somente em distribuições Linux e é originalmente compilado com gcc14.
